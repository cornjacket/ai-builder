## Goal

Add a policy to `roles/doc-format.md` and both ARCHITECT and IMPLEMENTOR prompts
stating that generated documentation must not cite specific tool/language version
numbers unless specifying a minimum required version constraint.

## Context

"Go 1.22 method+path patterns" — a specific version reference that will go stale.
The source of truth for versions is always a config file (go.mod, package.json, etc.).
